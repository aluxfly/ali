# 标书完整度检查功能说明

## 功能概述
在标书工厂中新增标书完整度检查功能，帮助投标人在生成标书后快速检查技术标和商务标的完整性，识别缺失项和潜在风险。

## 修改文件
- `bid_factory/index.html` - 添加检查按钮和结果展示区域
- `bid_factory/app.js` - 实现检查逻辑和可视化

## 功能特性

### 1. 检查按钮
- 位置：步骤 4（下载结果）页面，在下载按钮下方
- 样式：绿色主题按钮，标识为"📋 标书检查"

### 2. 技术标检查项（总分 100 分）
| 检查项 | 权重 | 说明 |
|--------|------|------|
| 施工方案 | 20% | 施工组织设计方案完整性 |
| 质量保证 | 15% | 质量管理体系和措施 |
| 安全措施 | 15% | 安全文明施工措施 |
| 进度计划 | 15% | 项目进度安排合理性 |
| 人员配置 | 15% | 项目团队人员配置 |
| 设备清单 | 10% | 施工/服务设备清单 |
| 技术方案 | 10% | 技术路线和系统设计 |

### 3. 商务标检查项（总分 100 分）
| 检查项 | 权重 | 说明 |
|--------|------|------|
| 投标函 | 15% | 正式投标函件 |
| 报价单 | 20% | 详细报价表格 |
| 资质证明 | 15% | 企业资质证书 |
| 业绩证明 | 15% | 类似项目业绩 |
| 财务报表 | 15% | 近三年财务审计 |
| 授权委托书 | 10% | 法定代表人授权 |
| 商务条款响应 | 10% | 商务条款响应表 |

### 4. 输出内容
- **完整度评分**：0-100 分，根据检查项完成情况计算
  - ≥90 分：绿色（优秀）
  - 70-89 分：橙色（良好）
  - <70 分：红色（需改进）
  
- **缺失项列表**：列出未完成的检查项

- **风险提示**：黄色警告框，提示缺失项可能影响评标得分

### 5. 可视化展示
- 技术标：蓝色主题，进度条 + 检查清单
- 商务标：红色主题，进度条 + 检查清单
- 总体评分：居中显示，大号字体

## 使用方法
1. 上传招标公告并生成标书
2. 在步骤 4（下载结果）页面点击"📋 标书检查"按钮
3. 查看检查结果，根据提示补充缺失内容
4. 下载标书文件
