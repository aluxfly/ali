## 💡 技术要点

### 问题本质
```javascript
// ❌ 错误：使用 /g 标志
const pattern = /项目名称\s*[：:]\s*(.+?)(?:\n|$)/gi;
text.match(pattern); 
// 返回：['项目名称：XXX 项目'] - 没有捕获组

// ✅ 正确：不使用 /g 标志
const pattern = /项目名称\s*[：:]\s*(.+?)(?:\n|$)/i;
text.match(pattern);
// 返回：['项目名称：XXX 项目', 'XXX 项目'] - 包含捕获组
```

### JavaScript match() 行为
- **有 /g 标志**: 返回所有匹配项的数组，**不包含捕获组**
- **无 /g 标志**: 返回第一个匹配项及其捕获组
