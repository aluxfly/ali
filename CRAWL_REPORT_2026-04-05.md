## 技术说明

### 爬取方法

1. 使用 Playwright 浏览器自动化
2. 访问国家电网 ECP 平台招标公告列表页面
3. 等待 JavaScript 渲染完成
4. 使用 JavaScript 评估器提取 DOM 数据
5. 智能过滤导航菜单项，只保留真实招标公告

### 数据字段

- `title`: 招标公告标题
- `link`: 详情页面链接
- `publish_date`: 发布日期
- `region`: 地区/省份
- `description`: 项目类型（招标/采购/谈判等）
- `crawl_time`: 爬取时间

### 环境配置

已安装系统依赖：
- atk, pango, cairo
- libX11, libXcomposite, libXdamage, libXext
- libdrm, libxcb, mesa-libgbm
- at-spi2-atk, at-spi2-core, alsa-lib
