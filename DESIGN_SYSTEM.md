# 项目雷达 - 视觉设计系统

## 📋 设计概述

本次设计美化旨在统一项目雷达三个功能模块的视觉风格，打造专业、简洁、一致的 B 端工具界面。

---

## 🎨 设计原则

1. **简洁专业** - B 端工具风格，避免过度装饰
2. **色彩统一** - 主色 + 辅助色不超过 4 种
3. **标识清晰** - 三个功能模块有视觉区分但风格统一
4. **体验一致** - 导航、按钮、卡片等组件保持统一交互

## 📝 使用说明

### 开发新页面
1. 引入 `css/common.css`
2. 使用预定义的组件类 (`.card`, `.btn`, `.form-input` 等)
3. 遵循模块配色规范

### 自定义模块色
```css
/* 在页面内联样式中覆盖 */
.navbar-logo {
  background: linear-gradient(135deg, #你的主色, #你的深色);
}
```

### 添加新组件
在 `css/common.css` 中按现有规范扩展，保持设计一致性。
