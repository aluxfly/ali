### 按钮交互流程

1. 用户点击"🔄 更新数据"按钮
2. 按钮变为黄色，图标开始旋转
3. 显示"正在更新数据"Toast 提示
4. 后端执行爬虫脚本（约 45 秒）
5. 显示"数据更新成功"Toast 提示
6. 3 秒后自动刷新页面
7. 显示最新数据

### API 调用示例

```bash
# 检查服务状态
curl http://localhost:3000/health

# 查看数据状态
curl http://localhost:3000/api/data-status

# 触发数据更新
curl -X POST http://localhost:3000/api/update-data
```

### 部署步骤

1. **推送到 Git 仓库**
   ```bash
   git add .
   git commit -m "feat: 添加更新数据按钮功能"
   git push
   ```

2. **部署到服务器**
   ```bash
   # SSH 到服务器
   ssh user@server
   
   # 拉取代码
   cd /path/to/project-radar
   git pull
   
   # 安装依赖
   npm install --production
   
   # 启动服务（使用 PM2 或 systemd）
   pm2 start server.js --name project-radar
   ```

3. **配置反向代理（Nginx）**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       
       location / {
           proxy_pass http://localhost:3000;
           proxy_http_version 1.1;
           proxy_set_header Upgrade $http_upgrade;
           proxy_set_header Connection 'upgrade';
           proxy_set_header Host $host;
           proxy_cache_bypass $http_upgrade;
       }
   }
   ```
