# 项目雷达 - 功能测试说明

## 新增功能（2026-04-05）

### 1. 日期选择器

**位置**：筛选区域第三列

**功能**：
- 点击日期输入框，选择特定日期
- 筛选出该日发布的所有项目
- 与其他筛选器（地区、类型、搜索）可组合使用

**测试步骤**：
1. 打开网站 https://aluxfly.github.io/ali/
2. 点击"发布日期"输入框
3. 选择一个日期（如 2025-12-20）
4. 观察项目列表更新，只显示该日期的项目
5. 统计信息显示筛选后的项目数量

### 2. 翻页功能

**位置**：项目列表底部

**功能**：
- 每页显示 20 条项目
- 显示总项目数和当前页码
- 支持上一页/下一页导航
- 支持直接点击页码跳转
- 大量页码时智能显示（如 1 ... 3 4 5 ... 10）

**测试步骤**：
1. 打开网站（61 条数据，应显示 4 页）
2. 观察底部翻页控件
3. 点击"下一页"，验证跳转到第 2 页
4. 点击页码"3"，验证直接跳转到第 3 页
5. 点击"上一页"，验证返回第 2 页
6. 观察页码高亮状态是否正确

### 3. 清除筛选

**位置**：筛选区域下方

**功能**：
- 一键清除所有筛选条件
- 重置到第 1 页
- 恢复显示全部项目

**测试步骤**：
1. 设置多个筛选条件（地区、日期、搜索词）
2. 点击"清除所有筛选"链接
3. 验证所有筛选框清空
4. 验证项目列表恢复全部显示

### 4. 组合筛选测试

**测试场景**：
1. 地区 + 日期：选择"北京" + 特定日期
2. 类型 + 搜索：选择"招标" + 搜索关键词
3. 全部组合：地区 + 类型 + 日期 + 搜索

**预期结果**：
- 筛选结果正确
- 翻页功能正常工作
- 统计信息准确

## 网站地址

- GitHub Pages: https://aluxfly.github.io/ali/

## 技术实现

- 每页项目数：20 条
- 日期筛选：精确匹配发布日期前缀（YYYY-MM-DD）
- 翻页逻辑：基于筛选后的项目列表计算
- UI 框架：Tailwind CSS

## 已知限制

- 日期筛选仅支持单日期选择（非日期范围）
- 翻页时自动滚动到列表顶部
